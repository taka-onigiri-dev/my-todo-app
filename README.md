# my-todo-app

# To-Doアプリ設計概要

## 基本機能

- **タスク管理**
  - タスクの追加、編集、削除
  - タスクのステータス管理（完了・未完了）
  - **タスクの並べ替え機能**: 個々のタスクを自由に並べ替え可能

- **カテゴリー管理**
  - タスクのカテゴリー分け
  - **カテゴリーの並べ替え機能**: カテゴリーを自由に並べ替え可能

## データ永続化

- **ローカルストレージ**: UserDefaultsを使用
  - **メリット**: シンプルで軽量、検索機能不要なシンプルなデータ管理に最適

## 技術スタック

- **UIフレームワーク**: UIKit
- **データ管理**: UserDefaults
