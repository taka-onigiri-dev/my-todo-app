# my-todo-app

# To-Doアプリ設計概要

## 基本機能

- **タスク管理**
  - タスクの追加、編集、削除
  - タスクのステータス管理（完了・未完了）
  - **タスクの並べ替え機能**: 個々のタスクを自由に並べ替え可能

- **カテゴリー管理**
  - タスクのカテゴリー分け
  - **カテゴリーの並べ替え機能**: カテゴリーを自由に並べ替え可能

## データ永続化

- **ローカルストレージ**: UserDefaultsを使用
  - **メリット**: シンプルで軽量、検索機能不要なシンプルなデータ管理に最適

## 技術スタック

- **UIフレームワーク**: UIKit
- **データ管理**: UserDefaults

## プロジェクト構成

```
MyTodoApp/

```

## 実装機能

### 1. タスク管理
- タスクの追加：タイトルとカテゴリーを指定して新規タスクを作成
- タスクの編集：既存のタスクのタイトルやカテゴリーを変更
- タスクの削除：スワイプまたは編集モードで削除
- 完了状態の切り替え：チェックボックスをタップして完了/未完了を切り替え
- ドラッグ＆ドロップで並べ替え：編集モードでタスクの表示順を変更

### 2. カテゴリー管理
- カテゴリーの追加：新しいカテゴリーを作成
- カテゴリーの編集：カテゴリー名を変更
- カテゴリーの削除：カテゴリーを削除（所属タスクは「未分類」に移動）
- ドラッグ＆ドロップで並べ替え：編集モードでカテゴリーの表示順を変更

### 3. データ永続化
- UserDefaultsを使用してタスクとカテゴリーのデータを保存
- アプリ起動時に自動的にデータを読み込み
- データ変更時に自動的に保存

## 使い方

このプロジェクトはSwift/UIKitで書かれたiOSアプリです。Xcodeで開いてビルド・実行してください。

1. Xcodeでプロジェクトを開く
2. シミュレータまたは実機を選択
3. ビルドして実行

## 必要要件

- iOS 13.0以上
- Xcode 11.0以上
- Swift 5.0以上
